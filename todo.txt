API Spec

GET /users/me/
    Get my user's data
POST /users/
    Create a new user
PUT /users/me/
    Update my user data

POST /auth/token/
    Login

GET /tasks/
    Get all of my user's tasks
POST /tasks/
    Creates a new task

GET /tasks/{id}/
    Get a specific task of mine
PUT /tasks/{id}/
    Updates a task
DELETE /tasks/{id}/
    Deletes a task

GET /tomatoes/
    Get all of my user's completed tomatoes
POST /tomatoes/
    Start a new tomato
